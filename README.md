# 線上程式碼剪貼簿

一個支援**本地儲存**和**線上共享**的程式碼剪貼簿工具，使用純 JavaScript 和 Firebase 開發。

## 🎯 核心功能

### 💾 本地功能
- **程式碼編輯** - 支援多種程式語言的語法高亮顯示
- **本地儲存** - 使用瀏覽器 localStorage 儲存程式碼片段
- **快速複製** - 一鍵複製程式碼到剪貼簿
- **自動儲存** - 防止意外遺失編輯內容

### 🌐 線上功能（新）
- **線上分享** - 將程式碼片段分享到雲端供他人瀏覽
- **即時載入** - 瀏覽和載入其他用戶分享的程式碼
- **搜尋功能** - 根據標題、內容或程式語言搜尋
- **統計資訊** - 顯示檢視次數和按讚數
- **語言篩選** - 依程式語言篩選線上程式碼片段

### 💻 支援的程式語言
- JavaScript
- HTML  
- CSS
- Python
- Java
- C++
- C#
- PHP
- SQL
- JSON

## 🚀 快速開始

### 方法 1: 僅使用本地功能
直接開啟 `index.html` 即可使用本地儲存功能。

### 方法 2: 啟用線上功能
1. 設定 Firebase 專案（請參考 [Firebase 設定指南](FIREBASE_SETUP.md)）
2. 更新 `index.html` 中的 Firebase 設定
3. 開啟網頁即可享受完整功能

## 📖 使用方法

### 基本操作
1. 在編輯器中輸入您的程式碼
2. 選擇對應的程式語言
3. 輸入片段標題（可選）
4. 選擇儲存方式：
   - **💾 儲存到本地** - 僅儲存在您的裝置上
   - **🌐 分享到線上** - 分享給所有用戶

### 瀏覽線上程式碼
1. 點擊 **📥 載入最新** 獲取最新的線上程式碼片段
2. 使用搜尋框尋找特定程式碼
3. 使用語言篩選器縮小範圍
4. 點擊 **📝 載入** 將程式碼載入編輯器

### 管理程式碼片段
- **載入** - 點擊程式碼片段的「載入」按鈕將其載入到編輯器
- **複製** - 點擊「複製」按鈕將程式碼複製到剪貼簿
- **刪除** - 只能刪除本地儲存的片段

### 快速操作
- **複製當前程式碼** - 點擊「複製」按鈕
- **清空編輯器** - 點擊「清空」按鈕
- **鍵盤快捷鍵**:
  - `Ctrl + S` - 儲存到本地
  - `Ctrl + Shift + C` - 複製程式碼
  - `Ctrl + Shift + L` - 清空編輯器
  - `Enter` - 在搜尋框中搜尋

## 🔧 部署選項

### 本地開發
```bash
python3 -m http.server 8000
# 或
npx serve
```

### GitHub Pages
1. 推送程式碼到 GitHub
2. 到 Repository Settings → Pages
3. 選擇 Source 為 `main` 分支
4. 設定 Firebase（如需線上功能）

### Firebase Hosting
```bash
firebase init hosting
firebase deploy
```

### 其他託管服務
- Netlify
- Vercel  
- Heroku（需要設定靜態檔案服務）

## 🔒 隱私與安全

### 本地模式
- ✅ **完全隱私** - 資料僅儲存在您的瀏覽器中
- ✅ **離線可用** - 無網路連線時仍可正常使用
- ⚠️ **清除資料注意** - 清除瀏覽器資料會遺失所有片段
